План розработки:
1. Дизайн приложения - готово
2. Верстка и оформление приложения(HTML&CSS) - готово
3. Создание базы данных, заполнить тестовыми данными - готово
4. Подключение базы данных
5. Выводить список контактов в чате 
6. При клике на пользователя с контактов выводить мою переписку с ним
7. Записовать сообщения в базу данных , введенные в чате.
8. Сделать регистрацию пользователя в базу данных с добавление в базу данных.
9. Проверять заригистрирован такой пользователь или нет.
8. Сделать авторизацию пользователя
9. Сделать поиск пользователяв контактах
10. Сделать модальное окно контакты
11. Сделать добавление и удаление друзей в модальном окне контакты.



Доработки:
1.Должно показывать активный(online) или не активный(offline) пользователь.
2. Прочитано сообщение или нет.
3. Отправка сообщения без перезагрузки страницы
4. Доделать выход с переадресацией

Структура папок и файлов:

Корневая папка :
 index.php - главная страница приложения
 add_friends.php - функционал для добавления пользователя в друзья
 delete_friends.php - функционал удаления пользателя в друзья
 exit.php - страничка для выхода , в котрой сбрасываються куки
 login.php - функционал для авторизации в чат 
 register.php - функционал для регистрации пользователя
 add_messages.php  - функционал добавления сообщений через AJAX

    /components - папка в которой находяться отдельные компоненты чата
         - header.php - функционал для шапки сайта , выход и вход.
    
    /config - папка в которой находяться настройки приложения
         - db.php - подключение к базе данных
         - settings.php - Прописываем разные настройки , чтобы легче было поменять значения

    /img - картинки, икноки приложения
     /js - файлы Java Script приложения
         - script.js - основной файл js
  /modules - папка в которой храняться функционал приложения
         - contacts.php - Модальное окно с контактами
         - list-messages.php - переписка пользователей
         - list-users - данные о пользователях приложения
  /style - папка со стилями приложения
         - style.css - главный файл для стилей
